HTML Conceitos

Introdução ao HTML:

O HTML é o conteúdo de um site,ou seja,não programamos em HTML,já que é apenas uma marcação de texto.

Primeiros passos:

No HTML5 existe um fonte de codígo base,existindo um atalho para colocar todo codigo base em apenas um sinal que no caso é <!>,assim abrindo o documento base.Assim tendo tambem algumas tags básicas que são :

  1. <Docktype> = HTML atualizado.
  2. <head>= Cabeça do site.
  3. <body>= Corpo do site.
  4. <meta>=Mantem o site em ordem e fornece dados.
  5. <lang>=Linguagem da página.
  6. <p>=Páragarfo.
  7. <h1>=Titulo dentro do site.
  8. <br>=Quebra de linhas.
  9. <hr>=Linhas horizontais.

Imagens e favicon:

Imagens:

Para inserir uma imagem no HTML, basta adicionar a tag <img src="sua imagem" alt "legenda">.Vale ressaltar que as imagens em PNG resultam em uma melhor resolução visualmente.

Favicon:

Para inserir a logo do seu site, você pode usar a tag Link:favicon, que ficará visualmente assim: <link rel="shortcut icon" href="Adiconar a sua logo" type="image/x-icon"> .

Formatação de texto:

A maior mudança do HTML4 para o HTML5 foi a acrescentação da semântica (significado ao vocabulário), sendo assim algumas tags acabaram morrendo para usarmos agora a tag com significado forte. Agora vamos ver algumas tags que morreram para ter significado :

  1. <b> virou <strong>=Negrito
  2. <I> virou <em>=Itálico
  3. <u> virou <ins>=Texto sublinhado.
  4. <strike> virou <del>=Frase desconsiderada.

Agora falaremos de outras tags de marcações de texto que são:

  • <small>=Texto menor.
  • <Mark>=Marca texto.
  • <sup>=x2
  • <sub>=H2O.
  • <abbr>=Abreviação das plvr.
  • <q>=Aspas com significado.
  • <code>=Codígos no computador.

Listas em HTML:

As listas que fazemos no nosso dia a dia são essenciais na vida de praticamente todas as pessoas. Um grande exemplo comum de listas é o uso de listas de compra do mês de supermercado, que são feitas para não esquecer nada do que falta em sua casa. Na programação, o uso de listas também é essencial.Agora iremos ver como funciona a utilização de listas em HTML:

Listas ordenadas:

A tag que usamos na lista ordenada é: <ol>,assim existindo type que são como você irá querer deixar sua lista: o type 1 significa a utilização de números, o type A significa letras maiúsculas,o type a significa minúsculas e o I que significa números romanos.

Listas não ordenadas:

A tag que usamos na lista não ordenada é <ul>,assim existindo outros tipos de type são o "disc", que é uma bola preta em formato grande, o "circle", que também é uma bola preta, só que pequena, e o "square", que é um quadrado perfeito.

Dentro das duas listas existe uma tag que se chama <li> que significa o conteúdo da linha (list item).

Lista de definição:

Utilizamos a tag <dl> para definir a lista,usamos a tag <dd>para definirmos o tema e por fim usamos a tag <dt> para colocar o termo que será definido.

Imagens dinâmicas, áudios e vídeos

Imagens dinâmicas:

Vamos relembrar que o site pode ser acessado em vários dispositivos, assim as imagens precisam ser compatíveis em tamanhos para todos os dispositivos como: celular, tablet e computador. Para isso, usamos o App GIMP e alteramos as imagens nos seguintes formatos:

  1. Celular: 300x300px
  2. Tablet: 700x700px
  3. Computador: 1000x1000px

A grande dificuldade é como saberemos onde cada pessoa está acessando o seu dispositivo: se é em seu celular, tablet ou computador. E para diferenciar as imagens para cada um deles, teremos que usar uma tag que se forma assim:

<picture <source media="maxímo de pixels" srcset="foto que adicionará caso passe do limite de pixels" type=image/png>

Estrutura basíca:

  • <!DOCTYPE html>: Declara o tipo de documento e a versão do HTML.
  • <html>: Define o elemento raiz de um documento HTML.
  • <head>: Contém metadados e links para recursos externos.
  • <title>: Define o título da página.
  • < meta>: Fornece metadados sobre o documento.
  • <link>: Liga o documento a um recurso externo, como uma folha de estilo.
  • <style>: Define estilos CSS internos.
  • <script>: Define scripts JavaScript.

Estrutura de Página:

  • <body>: Define o conteúdo principal do documento.
  • <header>: Define o cabeçalho da página.
  • <nav>: Define a navegação da página.
  • <main>: Define o conteúdo principal da página.
  • <footer>: Define o rodapé da página.
  • <section>: Define uma seção genérica em um documento.
  • <article>: Define um conteúdo independente e autossuficiente.
  • <aside>: Define conteúdo lateral, como barras laterais.

Formulário:

  • <form>: Define um formulário.
  • <input>: Define um campo de entrada.
  • <textarea>: Define uma área de texto multilinha.
  • <button> Define um botão.
  • <select>: Define uma lista suspensa.
  • <option>: Define uma opção em uma lista suspensa.
  • <label>: Define um rótulo para um elemento de formulário.
  • <fieldset>: Define um conjunto de campos de formulário relacionados.
  • <legend>: Define uma legenda para um elemento de campo de formulário agrupado.

Tabelas

  • <table>: Define uma tabela.
  • <tr>: Define uma linha em uma tabela.
  • <th>: Define um cabeçalho de célula em uma tabela.
  • <td>: Define uma célula em uma tabela.
  • <thead>: Define o cabeçalho de uma tabela.
  • <tbody>: Define o corpo de uma tabela.
  • <tfoot>: Define o rodapé de uma tabela.
  • <col>: Define as propriedades de uma ou mais colunas em uma tabela.
